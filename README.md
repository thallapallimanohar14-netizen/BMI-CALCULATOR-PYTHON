# 🧮 BMI Calculator using Python

A simple and interactive **Body Mass Index (BMI) Calculator** built using **Python**, **Tkinter**, and **Matplotlib**. This application calculates BMI, displays the corresponding health category, and provides a graphical visualization for better understanding.

---

# 📖 Overview

The BMI Calculator is a desktop application that helps users determine their Body Mass Index (BMI) based on their height and weight. It features an intuitive graphical interface, instant calculations, health status classification, and BMI graph visualization.

---

# ✨ Features

- ✅ User-friendly GUI built with Tkinter
- ✅ Accurate BMI calculation
- ✅ Health category classification
- ✅ BMI graph visualization using Matplotlib
- ✅ Input validation
- ✅ Simple and responsive design

---

# 🛠️ Technologies Used

- Python 3
- Tkinter
- Matplotlib

---

# 🧮 BMI Formula

```text
BMI = Weight (kg) / Height² (m²)
```

---

# 📊 BMI Categories

| BMI Range | Category |
|-----------|----------|
| Below 18.5 | Underweight |
| 18.5 – 24.9 | Normal Weight |
| 25.0 – 29.9 | Overweight |
| 30.0 and Above | Obese |

# 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/thallapallimanohar14-netizen/BMI-CALCULATOR-PYTHON.git
```

### Navigate to the Project Directory

```bash
cd BMI-CALCULATOR-PYTHON
```

### Install Dependencies

```bash
pip install matplotlib
```

### Run the Application

```bash
python bmi_calculator.py
```

---

# 💻 Usage

1. Enter your **Weight** in kilograms (kg).
2. Enter your **Height** in meters (m).
3. Click **Calculate BMI**.
4. View your BMI value and health category.
5. Click **Show BMI Graph** to visualize the result.
